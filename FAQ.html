Technology
  • What is CredX Coin (CDX)?

    CredX Coin (CDX) is a digital peer to peer network that aims to implement a use case specific blockchain, designed to efficiently handle one specific function: the transfer of assets from one party to another. Built on a fork of the Bitcoin code, CredX Coin (CDX) was launched 2nd October 2021, and is a truly open source project (no ICO). It focuses on building a useful technology, with a strong and growing community.

  • How CredX Blockchain works?

    A fork of the bitcoin code, To help promote Green Revolution there are some major changes done in CredX Blockchain. It features the issuance schedule (block reward of 0.00000001 CDX) and Block time (1 minute) and MasterNodes. Its coins generation system (mining system) is dual layered managed via MasterNodes and Master VM(Virtual Machine) to manage decentralised behaviour of Block Stacking. It is fully explained in WhitePaper Docs.

  • Does community members have any CDX?
    No there are no coins distributed to any of the developers or to any initial community members. Each and every one will have to go through the block stacking system to mine/get new coins. It will be distributed by the pre defined rules set in to the MasterNodes and Master VMs. Anyone can become block stackig member via contributing to it and start adding new coins to circulating supply.
  • How there is no carbon footprint of CDX?
    To attain decentralized etiquette we have formed a MasterNode and MasterVM concept. With the code of Bitcoin and Ethereum's Stacking Concept. So because there will be limited MasterNodes, there will be not be power usage in mining. And hence it is Green Revolution friendly coin with almost no carbon footprint. For proper understaing of it please read WhitePapers.
  • What is CredX Blockchain?

    It is transperant. It is just another bitcoin forked blockchain with intelligent code changes. More faster and light weight. With almost zero carbon footprint. The CredX blockchain is capable of handling higher transaction volume than its counterpart - Bitcoin. Due to more frequent block generation, the network supports more transactions without a need to modify the software in the future. As a result, merchants get faster confirmation times, while still having ability to wait for more confirmations when selling bigger ticket items.

  • What is CredX Wallet Encryption?

    Wallet encryption allows you to secure your wallet, so that you can view transactions and your account balance, but are required to enter your password before spending cdx(coins). This provides protection from wallet-stealing viruses and trojans as well as a sanity check before sending payments.

  • What is CredX Mining Rewards?

    Introducing BlockStacking. Its simple like ethereum and works on the basis of validator staking system. Stake to become a validator. Staking is a public good for the CDX ecosystem. You can help secure the network and earn rewards in the process.
